## Step-by-Step Guide

### Upload a New Version of a File

1. In your scheme's **Documents** tab, right-click a file and select **Add new version**
2. Select **Choose file** and browse to locate the file you want to upload
3. Select the file and confirm your selection
4. Select **Upload** and wait for the new version to complete uploading to Ctrl Hub

### Access Any Version of a File

1. In your scheme's **Documents** tab, right-click any file that has multiple versions
2. Select **History** from the context menu
3. In the Document Details panel, the currently selected version is displayed prominently at the top
4. Select any other version from the list to view its details

### View File Download History and Audit Log

1. In your scheme's **Documents** tab, right-click any document and select **Downloads**
2. The Download History panel will open for that document
3. View the complete download history showing details of all file downloads including who downloaded it, when, and which version was downloaded
4. This audit log provides a complete record of file access for compliance and tracking purposes
