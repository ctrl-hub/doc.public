## Step-by-Step Guide

### Set an Expiry Date for a File

1. In your scheme's **Documents** tab, right-click on the file you want to set an expiry date for
2. Select **Set expiry date** from the context menu
3. Choose an expiry date (you can select a past date if you want the file to display as expired immediately)
4. Select **Set Expiry** to apply the expiry date
