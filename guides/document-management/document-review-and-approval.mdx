## Step-by-Step Guide

### Request Document Review and Set Approval Deadlines

1. In your scheme's **Documents** tab, right-click any document and select **Request review**
2. In the **Approvers** section, select **Add** to add one or more approvers from the available names (your name is already included as an approver by default)
3. Select **Next**, skip the "deadline for comments" option, and select **Yes** for the "deadline for approval"
4. Set a due date and time for when approval is needed for this document
5. Select **Next** and optionally add a message with context or instructions for the approvers
6. Select **Request Review** to finish

### View Document Review and Approval Status

1. In your scheme's **Documents** tab, locate any document with a Review Status indicator
2. Click on the Review Status text (e.g., "Pending") next to the document to open the Document Review Requests panel
3. Alternatively, right-click the document and select **View review requests** to access the same panel
4. The panel displays the current approval status and which approvers have approved, pending, or rejected the document

### Access Document Review History and Approval Log

1. In your scheme's **Documents** tab, click on the Review Status for a document to open the Document Review Requests panel
2. Or right-click the document and select **View review requests** to open the panel
3. In the Document Review Requests panel, view the complete review history showing all approver decisions, comments, and timestamps
4. This log provides a complete audit trail of the approval process for the document

### Download a File

1. In your scheme's **Documents** tab, right-click any file you want to download
2. Select **Download** from the context menu
3. Your browser will offer you the option to Open or Save the file to your local device

### Access Documents for Shared Schemes as a Contractor

1. Go to **Governance > Schemes**
2. Select the tab corresponding to the organisation that has shared their schemes with you
3. Browse the shared schemes listed on this page
4. Select any shared scheme to access its documents and related information
5. You can view documents, check approval statuses, and download files as permitted
