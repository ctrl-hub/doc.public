## Step-by-Step Guide

### Assign operations to the workforce

1. Go to **Governance > Schemes**
2. If there is more than one organisation available, select the organisation who owns the Scheme
3. Select the required Scheme from the list
4. Select a Work Order from the Scheme
5. When the Work Order record opens, select an Operation from the list
6. When the Operation record opens, tick the checkbox next to the operation(s) you want to assign
7. In the **Actions** menu (at the top of the Operations checkbox column), select **Add**
8. In the **Add Assignees** window, enter an operative's name in the Search field and select **Search**
9. From the filtered list of names, select the operative you want to assign
10. Select **Add Assignees** to confirm the assignment
