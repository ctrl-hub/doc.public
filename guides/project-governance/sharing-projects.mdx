## Step-by-Step Guide

### Share a Scheme with a Contractor Organisation

1. Go to **Governance > Schemes** and select the scheme you created
2. Select the **Share** button in the top right of the page
3. If the scheme is available to share, select **Share** next to the contractor organisation name to want to share with
4. Select **Create Share** to confirm
