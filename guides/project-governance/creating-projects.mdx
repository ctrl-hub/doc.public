<Info>**Target Personas:** Project Manager, System Administrator, Scheme Owner</Info>

## Step-by-Step Guide

### Create a New Scheme

1. Navigate to **Governance > Schemes**
2. Select **Create a new Scheme**
3. In the Create Scheme window, enter a descriptive name for your scheme, for example: "GD3 - Main Route" or "Network Upgrade Phase 2"
4. Select **Create** to proceed

### Create a Work Order

1. Once the Scheme page opens, select **Create a new Work Order**
2. Enter a descriptive name that includes the location, for example: "Balcolme Road, Horley, RH6 9AY" or "Oxford Street Segment A"
3. Enter a unique code for the work order, for example: "WO-2026-001" or "SEO15242-A"
4. Select **Create** to finish

### Add Operations to Your Work Order

1. On the Work Order page, select **create multiple operations** (the link below the map)
2. When the Property search page opens, enter a full postcode and select **Search** (wait a moment for results to load)
3. From the available properties, select one or more properties and select **Next**
4. When prompted, in the Template field, select one or more required templates, for example: **Service Card**
5. Select **Finish** to create your operations
