## Step-by-Step Guide

### Update the Progress Status of an Operation

1. Go to **Governance > Schemes**
2. If there is more than one organisation available, select the organisation who owns the Scheme
3. Select a Scheme from the list
4. Select a Work Order from the Scheme
5. From the Work Order page, select an Operation from the list below the map
6. In the left-hand panel, in the **Status** section at the top, select the cog symbol
7. In the **Manage Status** window, select **In Progress**, **In Progress** or **Done**
8. If the work is **Done**, select a **Done reason**.
9. Select **Update Status** to save the change
