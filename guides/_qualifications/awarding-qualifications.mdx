<Info>**Target Personas:** HR Manager, Qualifications Manager, Compliance Officer, Training Administrator</Info>

## Step-by-Step guide

### Award a Qualification with a Certificate

1. Navigate to **People > Qualifications** and select the **Awards** tab
2. Find the qualification and holder you want to award
3. Select the **Award** button
4. In the file upload panel, select **Choose** and browse to locate a certificate file on your device
5. Select your file and confirm the selection
6. Select **Upload** and wait for the file to upload to Ctrl Hub
7. Select **Award** to finish
8. The holder's qualification will be updated with the Awarded date and an Expiry date calculated from the qualification's renewal period

### Set or Update an Awarded Date

1. Navigate to **People > Qualifications** and select the **Awards** tab
2. Find the qualification and holder you want to update
3. Select **Update** to open the award window
4. Set the 'start date of this award' to the appropriate date (this can be a past date if backdating an award)
5. Select **Update Award**
6. The Awarded date will be set to your specified date
7. The Expires date will be automatically calculated based on the qualification's renewal period from the awarded date
8. For example, if awarding a 5-year qualification on 1 December 2025, the Expires date will be 30 November 2030

### Edit a Qualification Award

1. Navigate to **People > Qualifications** and select the **Awards** tab
2. Find the qualification and holder you need to correct
3. Select **Update** to open the award window
4. Modify the information as needed (such as the awarded date, due date, or uploaded documents)
5. Select **Update Award** to save your changes
6. The holder's qualification Awarded and Expires dates will be recalculated based on your changes
