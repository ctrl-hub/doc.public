## Step-by-Step Guide

### Create Submissions While Offline

1. Once your device has sycned data, you can continue to work offline
2. Select an operation from your Operations list
3. Select the required form (such as "Service Card") from the Associated Forms section
4. Fill in the form completely, using the **Next** button to move between pages
5. If propmted, fill in any empty required fields
6. When you have completed all required sections of the form, select **Submit**
7. If prompted, fill in any empty further required fields, and select **Submit** again
8. The submission will be saved as "Pending" - awaiting a data connection for final submission

## What To Expect

After following these steps, you should have:

- Successfully installed and logged into the Ctrl Hub mobile app
- Access to all your assigned operations via the Operations page
- The ability to view daily operations on your home page
- Completed form submissions appearing in the Published list
- Draft submissions saved and available for continuation
- Understanding of offline functionality and how pending submissions are handled
- Successfully submitted forms that appear as "Published" in the Forms section
- Access to the web platform for viewing and tracking all scheme submissions
- The ability to filter submissions by Work Order for detailed analysis and compliance tracking
