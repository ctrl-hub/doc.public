## Step-by-Step Guide

### Continue and Complete a Draft Submission

1. On the Ctrl Hub app home page, locate **Recent Draft Form Submissions**
2. Select the draft submission you want to continue
3. Use the **Next** button to navigate through the submission to where you left off
4. Continue filling in the remaining required fields
5. When complete, select **Submit**
6. If prompted, review your form using **Previous** and **Next** buttons to identify and fill any empty required fields
7. Select **Submit** to finalise the submission

### Submit Offline Submissions When Connectivity Returns

1. Turn on your data connection (disable airplane mode or reconnect Wi-Fi/mobile data)
2. Open the Ctrl Hub app
3. Select **Forms** from the bottom toolbar
4. Select the **Submitted** tab to view your pending submissions
5. Wait a few moments for the app to attempt to submit any pending forms
6. Check the **Published** tab - your submission should now appear there with a status of "Published"
7. If your submission still appears as "Submitted", open it and select **Submit** again to retry
