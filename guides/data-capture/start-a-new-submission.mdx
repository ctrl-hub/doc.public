## Overview

This guide explains how to start a new submission using the Data Capture feature.

## Prerequisites

- Access to the Data Capture UI
- Appropriate permissions to create submissions

## Steps

1. Navigate to the Data Capture section.
2. Select "Start new submission".
3. Complete the required fields and submit.

## Expected Result

A new submission is created and appears in the submissions list.
