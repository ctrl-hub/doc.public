## Step-by-Step Guide

### Review Submissions in the Web Platform

1. Go to **Governance > Schemes** in the Ctrl Hub web platform
2. Select a scheme to view its details
3. On the Scheme page below the map, select the **Submissions** tab
4. View all submissions for all Work Orders and Operations under that scheme
5. Alternatively, select a specific Work Order and go to its **Submissions** tab to view submissions for only that work order
6. Review submission details, statuses, and metadata for compliance tracking
