<Info>**Target Personas:** New user, All users</Info>

## Step-by-Step Guide

### Accept the invitation received by email

1. Open the link in the email you will have received
2. From the Ctrl Hub web console login page, set your password
3. Log into the Ctrl Hub web console and open your profile by selecting the **profile icon** in the top right of the page
4. Select **Invitations**
5. You will see any pending invitations listed - select **Accept** for the organisation you want to join.
