## Step-by-Step guide

### Assign a Qualification to a Person

1. Navigate to **People > Qualifications** and select the **Qualifications** tab
2. Find the qualification you want to assign and select **Award**
3. In the 'Assign or award a qualification to someone' window, select **Assign**
4. In the **To** field, select the person to whom you're assigning this qualification
5. Leave the **Due Date** empty (you'll set this later if needed)
6. Leave the **Documents** field empty
7. Select **Assign** to finish
8. The person is now listed as a Holder of this qualification with a status of "No Risk"

### Set a Future Due Date for a Qualification

1. Navigate to **People > Qualifications** and select the **Qualifications** tab
2. Select the qualification you want to update
3. Find the assigned holder and select **Edit** from the Award dropdown button
4. Enter a future date in the **Due Date** field (this might represent when their training is scheduled)
5. Select **Update Assignment**
6. The holder's Expires date in the qualification list will be updated to reflect the Due Date you entered
