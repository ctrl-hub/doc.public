## Step-by-Step guide

### Renewing a Holder's Qualification

1. Navigate to **People > Qualifications** and select the **Awards** tab
2. Find a holder who has had their qualification renewed
3. Select the **Renew** option from the dropdown menu for that award
4. In the Renew Award window, set a renewal date
5. Expand the qualification record using the **>** symbol on the left side
6. The expanded view shows both the current award and any previous awards
7. Select **View History** to see the complete Award History for that holder's qualification, showing all awards, dates, and status changes

### Edit the Renewal Period and Recalculate Expiry Dates

1. Navigate to **People > Qualifications** and select the **Qualifications** tab
2. Select the qualification you want to update
3. Select **Edit** to open the qualification settings
4. Navigate to the Renewal settings section
5. Update the renewal period (in days) as needed
6. Select **Update** to save the changes
7. For all existing awards of this qualification, the Expiry dates will be automatically recalculated based on the new renewal period
8. For example, if you change a qualification from annual (365 days) to 3-year (1095 days) renewal, all existing holders' expiry dates will be extended by 2 additional years
