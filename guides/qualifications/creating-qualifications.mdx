<Info>**Target Personas:** System Administrator, Qualifications Manager, HR Manager</Info>

## Step-by-Step Guide

### Access the Qualifications Section

1. Navigate to **People > Qualifications**
2. Select the **Qualifications** tab
3. You'll see a list of existing qualifications in your system
4. Select **Create a new qualification** to begin creating a qualification

### Select a Qualification Type

1. On the Create Qualification page, click the **Qualification Type** dropdown list
2. You'll see a pre-populated list of available qualification types from industry standards
3. Select the qualification type that matches the qualification you're creating
4. When you select a qualification type, the **Name** field will automatically populate with the standard name

### Configure Basic Qualification Details

1. Review the auto-populated **Name** field - you can modify it if needed
2. Optionally, enter a **Description** to provide additional context about this qualification (for example: "Height Safety Training - Required for all site supervisors")
3. The description helps users understand the purpose and relevance of the qualification
4. Select **Next** to continue to the renewal settings

### Set Up Renewal Period

1. On the renewal configuration page, select **Renewable** to enable renewal cycles for this qualification
2. Enter the renewal duration in **days**:
   - For annual renewal: enter **365**
   - For 3-year renewal: enter **1095**
   - For 5-year renewal: enter **1825**
3. The system will use this duration to calculate when qualifications expire and renewals are due
4. Select **Next** to configure risk thresholds

### Configure Risk Thresholds

1. You'll see fields for **Low**, **Medium**, and **High** Risk thresholds
2. Leave the default values unless your organisation has specific risk assessment requirements
3. These thresholds determine how the system categorises qualification status based on time remaining until expiry
4. Select **Next** to proceed to assessment requirements

### Specify Assessment Requirements, where required

1. On the assessment page, you'll see the option **Requires assessment**
2. If this qualification requires an assessment (such as a CAS assessment), select **Yes**
3. In the **Reference** field, enter the assessment reference code (for example: "DN-14" for a specific CAS assessment)
4. Add any relevant notes, such as "1st job assessment" or other applicability criteria
5. These details help track which assessments are required for users holding this qualification

### Create the Qualification

1. Review all the settings you've configured
2. Select **Create Qualification**
3. The system will create your new custom qualification and return you to the Qualifications list
4. Your newly created qualification will appear in the Qualifications tab and is now available to assign to users
