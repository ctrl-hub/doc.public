## Step-by-Step Guide

### View Compliance Status for a Job Role

1. In the Ctrl Hub Web Console, navigate to **Competency &gt; Job Roles**
2. Select a Job Role from the list
3. Open the **Assigned &gt; Table** tab — all users assigned by contractors are listed here
4. In the **All organisations** dropdown, select a specific contractor organisation to filter the view to that organisation's competency data

<Info>The table shows each user assigned to the Job Role alongside their compliance status, which will display as either **Compliant** or **Not Compliant**.</Info>

---

### View the Competence Report for an Individual

1. From the **Job Role &gt; Assigned &gt; Table** view, locate a user whose compliance status you want to review
2. Select the compliance status label (either **Compliant** or **Not Compliant**) for that user — this opens the Compliance Report panel
3. Review the report to confirm the user's compliance detail:
   - **Not Compliant** means at least one **Required** competence requirement is not satisfied (shown with a red cross)
   - **Compliant** means all **Required** competence requirements are satisfied (shown with a green tick)
4. When finished, select the **X** in the top right corner of the panel to close it

<Tip>Only **Required** competence items affect overall compliance status. Optional items are visible in the report but do not impact the Compliant / Not Compliant result.</Tip>

---

### Review Competence Status Using the Matrix View

1. In a **Job Role &gt; Assigned** tab, select **Matrix** to switch from the Table view

<Info>The Matrix view displays the dates and status (tick or cross, with red or green backgrounds) for each person assigned to the Job Role. Clicking into any cell opens the detailed record for that specific competence requirement for the selected user.</Info>

---

### View an Evidence Certificate

1. From **Job Role &gt; Assigned &gt; Matrix** view, click into a cell for a **Training**, **Assessment**, or **Induction** requirement to open the details panel
2. Where evidence has been uploaded, select the file name to preview the document

<Tip>If the file type is not supported by your browser, you will be offered the option to download it instead.</Tip>

---

### Record a Verification Against an Awarded Competence

1. From **Job Role &gt; Assigned &gt; Matrix** view, select a **green** cell in the grid to open the details panel for an awarded competence item
2. At the bottom of the details panel, select **Record Verification**

<Info>Once submitted, your name and the date and time of the verification are recorded against the competence item. This confirms you have reviewed the record and are satisfied that the details are correct.</Info>

---

## What To Expect

After following this guide, you should be able to:

- View the compliance status of all users assigned to a Job Role, filtered by contractor organisation
- Open and review the individual Compliance Report for any assigned user
- Navigate the Matrix view to understand competence status at a glance
- Preview uploaded evidence certificates directly within Ctrl Hub
- Record a formal verification against any awarded competence record
