## Step-by-Step guide

### Upload Certificate Files

1. Navigate to **People > Qualifications** and select the **Awards** tab
2. Find the qualification and holder for which you need to upload a certificate
3. Select **Update** to open the award window
4. In the file upload panel, select **Choose** and browse for the certificate file
5. Select the file and confirm your selection
6. Select **Upload** and wait for the file to upload to Ctrl Hub
7. Repeat this process to upload additional certificate files (multiple certificates can be attached)
8. Close the window by selecting **Cancel** or the **X** button to finish
9. The certificate files are now associated with that holder's qualification record
