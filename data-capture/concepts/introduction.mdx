Data Capture is a relatively general phrase that refers to the process of capturing data from various sources and formats.

Usually when we discuss data capture in Ctrl Hub, it refers to "forms" which are completed and submitted. Our forms system allows you to build your own forms for users to fill out via the Ctrl Hub mobile app. Alternatively, you can also submit data via our API if you want more control over the experience.

If you want to integrate by calling our API, it is highly recommended to understand the technical architecture of the resources. There are some complex concepts that should be understood, particularly around schema evolution and submission versions.
